# Tokenizer (Lexical Analysis)
Tokens are pieces of strings which has identified meaning in a language.

Example code:
```

(add 1 2 (subtract 6 3))

```
Tokens are "(", "add", "1" , "2", "(", "subtract", "6", "3", ")", ")"
